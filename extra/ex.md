Exercício — Sistema de Pedidos da Cafeteria Interativa
Objetivo:

Criar um programa interativo que simule o atendimento de uma cafeteria, utilizando e em conjunto.

Requisitos obrigatórios:

Seu programa deve:

Mostrar uma mensagem de boas-vindas ao usuário.

Utilizar uma para permitir que o cliente faça , até decidir sair

Dentro do laço de repetição, o programa deve:
Perguntar o nome do cliente.
Perguntar qual bebida o cliente deseja.

Utilizar para:
Perguntar se o cliente deseja adicionar açúcar.
Se sim, mostrar uma mensagem confirmando.
Se não, mostrar uma mensagem diferente.

Perguntar se o cliente deseja um acompanhamento.
Se sim, perguntar qual e mostrar confirmação.
Se não, mostrar outra mensagem.

Ao final de cada pedido, perguntar se o cliente deseja fazer outro pedido.
Se sim, o programa continua.
Se não, o programa encerra.

Ao encerrar, mostrar uma mensagem de despedida.

O programa deve usar:
prompt() para entrada de dados
alert() para exibir mensagen
